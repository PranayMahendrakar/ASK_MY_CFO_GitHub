⚙ First-time setup (one-time)
  1. Push this project to GitHub: git push origin main
  2. Add your OpenAI key as a repo secret: Settings → Secrets → Actions → New → name: OPENAI_API_KEY
  3. Enable GitHub Pages: Settings → Pages → Source: GitHub Actions
  4. Create a Personal Access Token with repo + workflow scope
  5. Enter your repo name and token above, click Save

After setup, just drop PDFs and click "Upload & Run Pipeline" — everything runs on GitHub's servers for free.
